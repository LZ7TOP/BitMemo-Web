# BitMemo (像素记) 官方网站 📝

![Style](https://img.shields.io/badge/style-8--bit%20Pixel-orange)
![Framework](https://img.shields.io/badge/framework-React%20%2B%20Vite-blue)

**BitMemo (像素记)** 的官方展示与下载页面。这是一个极具复古 8-bit 像素画风的展示网站，旨在为用户提供沉浸式的极客体验。

---

## ✨ 官网特性

- **复古视觉 (Retro Visuals)**：完全遵照 8-bit 像素美学设计，包含硬核投影、粗边框以及经典的像素字体。
- **响应式布局 (Responsive)**：完美适配桌面与移动端，确保在任何设备上都能感受像素魅力。
- **动态交互 (Dynamic Interactions)**：充满灵动的像素动画效果，提升用户访问体验。
- **一键跳转 (Quick Access)**：提供浏览器插件的快速安装链接（即将上线）。

---

## 🛠️ 开发说明

本项目使用 **React + Vite** 构建，采用原生 CSS 进行样式定制，以确保最佳的性能与视觉还原。

### 本地运行

1. 克隆本项目：
   ```bash
   git clone https://github.com/LZ7TOP/BitMemo-Web.git
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```

### 生产构建

```bash
npm run build
```
